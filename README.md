#  Bookmark Saver

A simple and user-friendly **Bookmark Saver** application built using HTML, CSS, and JavaScript. This app allows users to save, view, and manage their favorite website links.


##  Features

*  Add bookmarks with name and URL
*  Clickable links for quick access
*  Dynamic bookmark list display
*  Option to delete bookmarks (if implemented in JS)
*  Clean and minimal user interface


##  Technologies Used

* HTML
* CSS
* JavaScript


##  How It Works

* Enter the **bookmark name**
* Enter the **website URL**
* Click **Add Bookmark**
* The bookmark appears in the list
* Click the link to visit the website
